# Postmortem Template

Use this when Analysis OS is applied retrospectively to an incident.

```
## Summary
<one paragraph: what happened, impact, duration>

## Timeline
<see prompts/timeline.md format>

## Root Cause
<see root-cause.md>

## What went well
<detection speed, mitigation effectiveness, etc — be specific, not generic>

## What went wrong
<be specific and blameless — process/system gaps, not individual blame>

## Action items
<see decision.md format, one per action item, each with an owner and due date>

## Unknowns / follow-up investigation needed
<anything Analysis OS's Explain phase couldn't fully resolve>
```
