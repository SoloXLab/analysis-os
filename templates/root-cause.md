# Root Cause Template

```
Root cause:              <the validated cause, not just a hypothesis>
Causal chain:             <A causes B causes C ... the observed symptom>
Confidence:                <high / medium / low + why>
Evidence:                  <the evidence that promoted this from hypothesis to cause>
Alternative causes ruled out: <list + why each was rejected>
Residual uncertainty:     <what's still not 100% certain>
```
